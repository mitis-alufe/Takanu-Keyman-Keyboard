Takanu (Lentan) keyboard
Based on Sitelen Pona (Lentan, UCSUR) keyboard
==============

Version 1.0

Description
-----------
This is a keyboard for typing Takanu, an unnoficial logographic
system for the language Kokanu. This keyboard uses the spelling of
words in Latin script to enter Takanu logographs.
